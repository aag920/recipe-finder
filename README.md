# Recipe Finder App

A recipe finder app built with React that allows users to search for recipes, view detailed information, and save their favorites

## Features

- Search recipes by name
- View detailed recipe information
- Save favorite recipes
- Watch video tutorials for recipes
- Responsive design

## Installation

1. Make sure you have Node.js installed

2. Clone or download this project

3. Navigate to the project directory:

   cd recipe-finder

4. Install dependencies:

npm install

5. Start the development server:

   npm start

6. Open your browser and visit `http://localhost:3000`

## Usage

1. Search for recipes: Enter a dish name in the search bar
2. View details: Click on any recipe card to see details, ingredients, and instructions
3. Save favorites: Click on the heart icon to add recipes to your favorites list
4. View favorites: Click the Favorites button to see all of your saved recipes
5. Watch tutorials: If available, click the video link to watch a tutorial

## Methods Used

- React
- JavaScript
- TheMealDB API (free recipe API)
